  3. Hapus semua kode yang ada, paste kode dari file apps-script.gs yang ikut di-download
  4. Klik Deploy → New Deployment → pilih type Web App
  5. Set Execute as: Me dan Who has access: Anyone → klik Deploy
  6. Copy URL Web App yang muncul, paste di bawah ini lalu klik Simpan & Test
